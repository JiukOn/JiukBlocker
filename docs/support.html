How You Can Help

JiukBlocker is a free project maintained with passion. If the extension has improved your browsing experience and you want to support our continuous updates, here is how you can help:

  • ⭐ Star the project: On GitHub or the Chrome Web Store to help the algorithm recommend us to more people.
  • 🐛 Report Bugs: Found something wrong? Open an Issue on GitHub. Keeping the rules updated is a community effort!
  • 📢 Spread the word: Recommend the extension to friends and family.

Financial Support

If you wish to financially support the development, any contribution is strictly **voluntary** and will be used for project maintenance, infrastructure, or simply motivation for new lines of code.
