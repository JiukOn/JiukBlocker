Como Você Pode Ajudar

O JiukBlocker é um projeto gratuito mantido com paixão. Se a extensão melhorou sua experiência de navegação e você quer apoiar nossas atualizações contínuas, aqui está como você pode ajudar:

  • ⭐ Dê uma estrela no projeto: No GitHub ou na Chrome Web Store para ajudar o algoritmo a nos recomendar para mais pessoas.
  • 🐛 Reporte Bugs: Encontrou algo errado? Abra uma Issue no GitHub. Manter as regras atualizadas é um esforço comunitário!
  • 📢 Espalhe a palavra: Recomende a extensão para amigos e familiares.

Suporte Financeiro

Se você deseja apoiar financeiramente o desenvolvimento, qualquer contribuição é estritamente **voluntária** e será usada para manutenção do projeto, infraestrutura ou simplesmente para um café para motivar novas linhas de código.
