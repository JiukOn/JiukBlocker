Bem-vindo ao JiukBlocker

A extensão definitiva para uma navegação limpa, segura e ultra-veloz.

O que é o JiukBlocker?

O JiukBlocker é uma extensão de navegador poderosa e altamente otimizada para o ecossistema Chromium, focada na filtragem contínua de conteúdos indesejados. Ele combina bloqueio de anúncios robusto com filtragem inteligente de conteúdo explícito (NSFW) usando análise dinâmica de DOM e um dicionário centralizado de heurísticas.

Foco em controle de conteúdo leve e cirurgicamente preciso, garantindo uma experiência de navegação discreta sem sacrificar o desempenho.

Como Funciona?

Nossa mágica reside na combinação de interceptação passiva de rede e varredura visual reativa:

  • Bloqueio de Rede Preventivo: Utiliza a API declarativeNetRequest para bloquear anúncios e rastreadores antes mesmo do carregamento.
  • Injeção Inteligente: Scripts específicos para plataformas como YouTube, Instagram, X (Twitter), TikTok e Reddit.
  • Escaneamento de DOM Otimizado: Monitoramento via MutationObserver para detectar novos conteúdos dinâmicos instantaneamente.
  • Ação Visual: Aplicação de filtros CSS como backdrop-filter: blur ou display: none em tempo real.

Estrutura do Projeto

O design é estritamente modular, facilitando a escalabilidade e manutenção:

  • /rules: Regras de rede para anúncios e NSFW.
  • /content: Lógica de front-end para filtragem em sites específicos.
  • /popup: Interface do usuário moderna para controle total.
  • /background: O cérebro que gerencia estados e regras dinâmicas.
