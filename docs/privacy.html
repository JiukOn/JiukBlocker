Política de Privacidade

Sua privacidade é nossa prioridade absoluta.

Transparência Total

O JiukBlocker é uma extensão aberta e transparente, projetada estritamente para melhorar sua experiência de navegação ao bloquear anúncios e distrações visuais. Respeitamos profundamente sua privacidade.

1. Coleta de Dados

Nós não coletamos, armazenamos, transmitimos ou vendemos quaisquer dados pessoais, histórico de navegação na web ou atividade do usuário. O projeto é auditável e focado em privacidade.

Nenhum dado sai da sua máquina.

2. Armazenamento Local

Todas as suas preferências (como palavras-chave bloqueadas e estados de alternância para anúncios, spam e shorts) são salvas localmente no seu dispositivo usando a API nativa chrome.storage.sync do Google Chrome.

  • Os desenvolvedores não têm **absolutamente nenhum acesso** a esses dados.
  • Suas configurações permanecem sob seu controle total.

3. Justificativa de Permissões

A extensão requer acesso às páginas da web (<all_urls>) estritamente para realizar modificações locais e imediatas no layout do site (DOM) para ocultar elementos intrusivos e anúncios.

  • Todo o processamento acontece inteiramente dentro do seu navegador.
  • O código é de código aberto para que qualquer pessoa possa verificar essas alegações.
