Guia de Contribuição

Ajude-nos a tornar o JiukBlocker ainda mais inteligente e eficiente.

Onde Começar?

Seja você um desenvolvedor experiente ou alguém querendo apenas sugerir uma nova regra de bloqueio, seguir este guia facilita o processo.

  • Dicionário de Filtros: Adicione novas palavras-chave de gatilho.
  • Suporte a Plataformas: Melhore o bloqueio em redes sociais específicas.
  • Interface: Melhore o menu da extensão e a UX.

Passos Técnicos

  1. Faça um Fork do repositório no GitHub.
  2. Crie uma branch para sua funcionalidade: git checkout -b feature/NovaFuncionalidade.
  3. Implemente suas mudanças seguindo os padrões de código do projeto.
  4. Envie um Pull Request detalhando o que foi alterado.

Adicionando um Novo Filtro

Para adicionar um novo termo ao dicionário NSFW global:

  • Abra o arquivo content/General/nsfw_keywords.js.
  • Adicione seu termo ao array exportado.
  • Todas as integrações da extensão ganharão acesso a essa atualização instantaneamente!

Expansão de Redes Sociais

Quer adicionar suporte nativo a um novo site (ex: Pinterest)?

  • Crie um diretório em content/Pinterest/.
  • Adicione a lógica em content_pinterest.js e os estilos em styles_pinterest.css.
  • Registre o novo padrão de URL no manifest.json.
